## Interactive Elements

### Search Input
- **Behavior**: Filters clients in real-time
- **Triggers**: On input, on focus
- **Shows**: Dropdown with max 15 results
- **Hides**: When user clicks away or selects

### Create Button
- **Visibility**: Only when search returns 0 results and user typed 2+ chars
- **Action**: Opens modal dialog
- **State**: Always available (unless creating in progress)

### Modal
- **Open**: Click "Create New Client" button
- **Close**: Click Cancel, X button, or outside backdrop
- **Submit**: Click Create (enabled only when name + phone filled)
- **Loading**: Shows spinner during submission

### Selected Client Card
- **Display**: Shows selected client info
- **Actions**: "Change" button to start new search
- **Persistence**: Stays selected until user clicks "Change"
